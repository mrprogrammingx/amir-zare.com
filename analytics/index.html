Analytics Platform

A full-stack analytics platform that normalizes telemetry, exposes fast analytical queries, and provides reproducible dashboards and reporting APIs.

Project description

This project collects streaming and batch telemetry, normalizes events into analytical tables, creates pre-aggregations, and exposes both dashboards (Streamlit) and programmatic endpoints (FastAPI) for reporting. Emphasis on reproducibility, testing, and low operational cost using DuckDB for analytical workloads.

Key features

  • Unified ingestion for JSONL telemetry and batch uploads
  • Automated schema evolution & data validation
  • Pre-aggregations and materialized views in DuckDB for fast queries
  • REST API for aggregated metrics and adapters for BI tools
  • Testing and CI for transforms to prevent regressions
